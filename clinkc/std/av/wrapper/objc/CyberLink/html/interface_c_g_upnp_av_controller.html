Detailed Description

The CGUpnpControlPoint class is a wrapper class for CgUpnpControlPoint of CyberLink for C to program using only Objective-C directly on MacOSX. Currently, the only basic methods are wrapped to control UPnP devices.

Member Function Documentation

- (NSArray *) servers

Activate some background threads of the control point such as SSDP and HTTP servers to listen messages and events of UPnP. You must call this method before you can actually use a control point.

Returns:
TRUE if successful; otherwise FALSE
